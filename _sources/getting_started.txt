Getting Started
================

Introduction
-------------

ForcedChoice is a Moa based package for running the CPL conditioning
experiment in the vet school.

Usage
------

To use CPLCom, you first need to install it, see :ref:`install-forced_choice`.

After it's installed, it is run through the main module. For example::

    python -m forced_choice.main

Complete API documentation is at :ref:`forced_choice-root-api`.
