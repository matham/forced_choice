Installation¶

Dependencies¶

  • Python 2.7+
  • CPLCom
  • FFPyPlayer
  • PyBarst
  • numpy (pip install numpy)

ForcedChoice¶

After installing the dependencies ForcedChoice can be installed using:

pip install https://github.com/matham/forced_choice/archive/master.zip
